Introduction

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python’s elegant syntax and dynamic typing, together with its interpreted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.

The Python interpreter is easily extended with new functions and data types implemented in C or C++ (or other languages callable from C). Python is also suitable as an extension language for customizable applications.

Hello World

To get started with Python, you're going to write a simple line of code to write a phrase on your screen.

  • Write the line: print('Hello World!')
  • The output should be: Hello World!

    • Variables

    A variable is used to store information in the memory, whether that be a string, a number, a boolean and so on.

    Usually a variable is written in all lower letters, and in case of two words or more, only the first should be written with lower letter, and no spacing.

  • Example: myVar = 5

    • And you can write the variable value with: print(myVar)

  • Output: 5

    • Input

    The command "input" allow the user to, as it says, input any keys depeding on their type.

    By writing only input('Write something here!'), it will allow the user to use any type of value.

  • It is possible to restrict to some by writing str(input('Write a string here!')), as this will only allow to input a string.
  • There is also int that only permits integer numbers, and float for float numbers. If the user try to write a string with the restriction of any of these two types, it will cause an error.

    • if Statements

    if Statements puts some conditions on the program that allows differents paths to run it.

    Here is an example: 

      myVar = 5
    if myVar > 1:
      print('It is greater than one.')
    else:
      print('It is not greater than one.')
  • The output will be: 'It is greater than one'.
  • If myVar is 1 or less, the output would be 'It is not greater than one.'

    • Loops

    There are two main commands of loops in Python: for and while.

    A "for" loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

    With the "while" loop we can execute a set of statements as long as a condition is true.

    Examples of each: 

          
   myVar = 0
   for number in range(1, 6):
     myVar += number
   print(myVar)
  • This code makes so that every number in range from 1 to 5 (the last number in the command does not count) adds up to myVar variable, so the output will be 15. 
        
   myVar = 0
   while True:
    myVar += 1
    if myVar == 5:
      break
   print(myVar)
  • This loop example makes myVar variable add + 1 while it is not equal to 5, so when it meets the necessary condition, it breaks the loop, and then print myVar with the output 5.
